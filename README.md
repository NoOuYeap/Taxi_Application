# 사업 전망
1. 차별성
인파 흐름이 집중되는 구간에서 경로 유사도를 기반으로 이용자를 자동 매칭하고, 승하차 허용 지점 중심으로 이동합니다.

# 사업화 추진 전략
1. 단기 시범 서비스 기간
인파가 많은 지역을 이용해 단거리 이용 수요 데이터 수집, 출발지와 도착지의 패턴 및 군집 분석, 매칭 성공률과 유사도의 정확도 향샹

2. 서비스 고도화 및 확장
AI 기반 경로 유사도 모델 적용, 혼잡도 예측 및 사용자 대기시간을 예측하는 고도화 진행

# 수익구조
1. 광고 및 프로모션
2. 지하철이나 버스 환승센터 주변 상권과 제휴 수익
3. 데이터 기반 교통 연구 및 지자체와의 협력 사업
4. 프리미엄 매칭 기능 (출발지 우선권, 특정 시간대 고정 매칭 기능)

# 시장성 및 기대효과
1.대규모 행사를 통한 사업 기반 확보 용이
연간 대형 행사의 수는 약 8000회 이상하다.
경기장 주변 대기시간은 평균 40분 이상이며, 수요가 반복되고 집중되어 데이터 기반 운영에 최적화 되어있다.
